🧬 Retrotransposones SVA

Genómica Epigenómica Oxford Nanopore TFM

Este proyecto desarrollado durante mi Trabajo Fin de Máster se centró en el estudio genómico y epigenómico de retrotransposones SVA en pacientes con deficiencia de antitrombina mediante la tecnología de Oxford Nanopore. A lo largo del análisis empleé herramientas bioinformáticas especializadas como Sniffles para la detección de variantes estructurales y RepeatMasker para la identificación de elementos repetitivos, logrando filtrar y caracterizar inserciones específicas responsables de la enfermedad.

🎯 Objetivos

  • Identificar inserciones de retrotransposones SVA mediante secuenciación Nanopore
  • Caracterizar patrones de metilación en regiones de inserción
  • Caracterización de la estructuras de los retrotransposones

🛠️ Metodología

  • Procesamiento de lecturas largas, basecalling
  • Alineamiento con minimap2
  • Detección de variantes estructurales con Sniffles
  • Análisis de metilación con Methylartist
  • Anotación de repetitivos con RepeatMasker

📊 Resultados Clave

  • Confiramción de identificación de inserciones SVA patogénicas en el gen SERPINC1
  • Caracterización del patrón de metilación diferencial
  • Desarrollo de pipeline automatizado para análisis

🔬 Tecnologías Utilizadas

  • Oxford Nanopore PromethION
  • Python, Bash, R para análisis
  • SLURM para HPC
  • IGV para visualización
  • Git para control de versiones

🌱 Proyecto Momentum - Ecosistema Digital para Mejora Genética

Smart Breeding Transcriptómica IA CEBAS-CSIC

Proyecto de investigación que busca establecer un ecosistema digital integral para aplicar técnicas de smart breeding en frutales. Integra el análisis de Big Data, desarrollo de software científico y uso de infraestructuras tecnológicas avanzadas. El proyecto incorpora tecnologías emergentes como Inteligencia Artificial, Internet of Things (IoT) y blockchain para mejorar la toma de decisiones y la eficiencia en programas de mejora genética vegetal.

🎯 Objetivos del Proyecto

  • Crear un repositorio web centralizado de datos genómicos, transcriptómicos y fenotípicos
  • Desarrollar base de datos relacional para gestión de germoplasma
  • Implementar pipelines de análisis ómico automatizados
  • Integrar datos de sensores IoT para fenotipado
  • Aplicar modelos de IA para predicción de rasgos

🔧 Mi Rol y Responsabilidades

  • Diseño y desarrollo del repositorio web con Flask
  • Implementación de base de datos PostgreSQL
  • Análisis de datos genómicos y transcriptómicos
  • Documentación técnica y científica

💻 Stack Tecnológico

  • Backend: Python, Flask, SQLAlchemy
  • Frontend: HTML, CSS, JavaScript, Tailwind
  • Base de Datos: PostgreSQL, MySQL
  • Análisis: R, Python (pandas, numpy, scikit-learn)
  • Infraestructura: Docker, Git, Linux

📈 Impacto Esperado

  • Optimización de recursos en programas de mejora

▶ 🔍 Contexto: ¿Qué es el Smart Breeding?

El smart breeding representa la evolución de los programas tradicionales de mejora genética mediante la integración de tecnologías digitales avanzadas. Permite analizar e interpretar grandes volúmenes de datos genómicos, fenotípicos y ambientales para realizar predicciones más precisas sobre el valor genético de los individuos.

La implementación requiere capacidades en análisis de Big Data, desarrollo de software en múltiples lenguajes y frameworks, gestión de infraestructuras tecnológicas (redes, servidores, cloud computing), y experiencia con tecnologías emergentes como IoT para monitorización ambiental, realidad virtual y aumentada para visualización de datos, blockchain para trazabilidad, e inteligencia artificial para modelado predictivo.
